Fullstack Development

Course

Learn to build full-stack web applications using .NET—a free, open-source, cross-platform framework for creating modern apps and powerful cloud services, while integrating frontend technologies using HTML, CSS, and JavaScript.

Full-stack engineers are responsible for both the server-side and client-side of an application. This program will focus on teaching you how to write backend code using C# and .NET, as well as integrating it with frontend technologies.
Prerequisite:
* There is no coding experience required for this course but participants must be computer literate and know how to navigate their way around it.
* A laptop with at least 4GB ram
* Passion for coding
Training Methodology:
Multiple training delivery methods are employed including Lectures, Demonstrations, Practical, Self Practice and workshop using real life case study with particular reference to Nigerian situations.
Duration:
* Six (6) months, twice a week — Mondays and Wednesdays (frontend 10am to 1pm, backend 12 to 2pm).

Frontend Course Contents:
- Introduction to Web Development/Html
- Styling HTML web pages Overview of CSS
- Intermediate CSS Crafting Responsive Layouts
- Bootsrap and responsive Design Introduction to Bootsrap
- Javascript Fundamentals and DOM
- JavaScript HTML DOM
- ES6 and JavaScript Modules New Variables
- JavaScript Objects
- JavaScript Functions
- JavaScript Classes
- Asynchronous JavaScript
- JavaScript Objects (ii)
- JavaScript Browser (BOM)
- JavaScript Web APIs
- JavaScript AJAX
- JavaScript JSON
- Final/Capstone Project
